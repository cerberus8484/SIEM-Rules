## Generating Current Coverage

```bash
python tools/coverage_matrix.py          # writes COVERAGE.md + coverage.json
python tools/coverage_matrix.py --stdout # print to console without writing files
python tools/coverage_matrix.py --md-only   # skip coverage.json
python tools/coverage_matrix.py --json-only # skip COVERAGE.md
```

If CI reports a coverage diff, regenerate and commit:

```bash
python tools/coverage_matrix.py
git add COVERAGE.md coverage.json
git commit -m "chore: update coverage matrix"
```
